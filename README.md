# Agentic-OS: predice que tipo de usuario eres: 
# Compilación

Desde la carpeta raíz ejecutar:

```bash
make
```

Para eliminar los ejecutables:

```bash
make clean
```

---

# Ejecución

## Terminal 1

Iniciar el servidor IA Learner.

```bash
./bin/ialearner
```

o

```bash
cd bin
./ialearner
```

---

## Terminal 2

Iniciar el launcher.

```bash
./bin/launcher
```

o

```bash
cd bin
./launcher
```
